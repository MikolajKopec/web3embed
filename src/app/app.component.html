General Product Questions

What is Web3Checkout and how does it work?

Web3Checkout is a payment processing solution that allows businesses to accept cryptocurrency payments easily. Built on Supabase and FastAPI, it provides a secure, reliable foundation for processing blockchain transactions.

The system works by generating embedded payment components that can be added to any website with a simple code snippet. When customers make payments, transactions are verified on the blockchain, and notifications are sent to both merchant and customer.

How does Web3Checkout compare to other crypto payment solutions?

Web3Checkout differentiates itself in several key ways:

  • No KYC Requirements: Accept payments without requiring customers to go through identity verification processes
  • Flexible Offer System: Create multiple payment offers with different tokens and networks based on your needs
  • Simple Transactions: Direct wallet-to-wallet payments without complex return/refund mechanisms
  • Zero Transaction Fees: We charge only a flat monthly fee, not per-transaction fees
  • Integration Simplicity: Implement with just a few lines of code, no deep blockchain knowledge required

Our focus is on making cryptocurrency payments as simple as possible for both merchants and customers, removing traditional payment friction without sacrificing reliability.

Technical Integration

How long does a typical integration take?

Basic integration of Web3Checkout can be completed in as little as 10-15 minutes. The process involves:

  1. Creating an account in our dashboard
  2. Setting up your first payment offer
  3. Copying your unique embed code
  4. Pasting it into your website's HTML

For more complex integrations using our API or advanced webhook configurations, implementation might take 1-2 hours depending on your specific requirements.

Do I need to understand blockchain technology to use this service?

No, you don't need any blockchain or cryptocurrency expertise to use Web3Checkout. Our platform is designed for ease of use, handling all the complex blockchain interactions behind the scenes.

You simply need to decide which cryptocurrencies you want to accept and embed our component on your website. We handle all blockchain-related complexities including network selection, token standards, transaction verification, and more.

How does the Supabase integration work?

Web3Checkout is built on Supabase, a powerful open-source Firebase alternative. Our Supabase integration provides:

  • Authentication: Secure user authentication for your merchant dashboard
  • Database: Reliable PostgreSQL database for storing transaction data
  • Real-time Updates: Instant transaction status updates via Supabase's real-time capabilities
  • Row-Level Security: Ensures merchants can only access their own data

Our FastAPI backend works with Supabase to provide high-performance transaction processing and verification.

Payment Processing

What cryptocurrencies and networks can be accepted with Web3Checkout?

Web3Checkout supports all cryptocurrencies and networks that are compatible with MetaMask, including:

  • Major Tokens: ETH, USDC, USDT, DAI, WETH, WBTC, and hundreds more
  • Networks: Ethereum Mainnet, Sepolia Testnet, Polygon, Arbitrum, Optimism, zkSync Era, and more
  • Token Standards: Native tokens, ERC-20, ERC-721, and more

Our flexible offer system allows merchants to specify which tokens and networks they want to accept for each payment offer. You can add any token available in MetaMask to your accepted payment methods.

How does payment verification work?

Web3Checkout uses a multi-stage verification process to ensure payment integrity:

  1. Transaction Submission: Customer submits payment via MetaMask
  2. Pending Status: Transaction is marked as "pending" and tracked by our system
  3. Blockchain Confirmation: We monitor the blockchain for confirmation of the transaction
  4. Amount Verification: We verify that the correct amount was sent to the correct address
  5. Status Update: Transaction is marked as "verified" or "failed" based on blockchain confirmation
  6. Notification: Webhooks and emails are sent to notify relevant parties

This process ensures that only legitimate, confirmed transactions are processed, protecting both merchants and customers.

How does the offer system work?

The offer system is a core feature of Web3Checkout that allows you to create multiple payment configurations:

  • Multiple Offers: Create different payment options for different products or services
  • Token Selection: Specify which cryptocurrencies you accept for each offer
  • Network Selection: Choose which blockchain networks are supported
  • Custom Pricing: Set specific price points in USD that automatically convert to token amounts
  • Expiration: Set deadlines for payment offers

Each offer can be embedded separately on different pages or products, giving you full flexibility in how you accept cryptocurrency payments.
